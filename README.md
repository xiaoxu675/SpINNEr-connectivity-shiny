# Shiny App for Connectivity Visualization

This repository contains a Shiny application for interactive visualization of bootstrap-stable functional connectivity results from our study.

## Overview

The app allows users to explore connectivity patterns across networks and conditions, including differences between high- and low-sucrose conditions and stability thresholds.

## How to run

Open R and run:

```r
shiny::runApp()
```
## Data

The repository includes processed summary data used for visualization. Raw neuroimaging data are not publicly available due to privacy and data-use restrictions.

## Notes

This Shiny app is provided as a companion visualization tool to support the interpretation of results reported in the manuscript.
